# Projekt: Konfiguration einer Umgebung für die IKR SimLib
In diesem Projekt wurde das Programm IKR SimLib der Universität Stuttgart so konfiguriert, dass Studenten oder Dozenten der THI es einfach benutzen können.
In der Datei "Dokumentation zu SimLib.pdf" ist eine detaillierter Schritt-für-Schritt Anleitung für den Einstieger erstellt worden.
In der .zip Datei "zusaetzliche Dokumente" befinden sich weiterführende Dokumente, für all diejenigen, die sich noch intensiver mit dem Programm beschäftigen wollen.
Ebenfalls weitere Dokumente sind unter [http://www.ikr.uni-stuttgart.de/Content/IKRSimLib/] zu finden.
